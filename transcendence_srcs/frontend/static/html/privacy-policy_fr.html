Politique de Confidentialité

Dernière Mise a Jour : 09/01/2025

Chez Transcendence_Incorporation, nous accordons une grande (peu) importance à la protection de vos données personnelles. Cette Politique de confidentialité vous informe sur la manière dont nous collectons, utilisons et protégeons vos données, conformément au Règlement Général sur la Protection des Données (RGPD) et aux lois applicables.

1. Responsable du traitement

2. Données personnelles collectées

Nous collectons les données personnelles suivantes :
  • Données d’identification : nom, prénom, adresse e-mail, etc.
  • Données de connexion : adresse IP, type de navigateur, langue du navigateur, etc.
  • Autres informations : le nom de votre animal de companie.

3. Finalités du traitement

Nous utilisons vos données personnelles pour :
  • Fournir et gérer nos services (exemple : création de compte, création de tournois).
  • Améliorer notre site web ou nos services grâce à des analyses statistiques.
  • Respecter nos obligations légales et réglementaires.

4. Base légale du traitement

Les traitements effectués reposent sur :
  • L’exécution d’un contrat (article 6(1)(b)) : pour fournir les services que vous avez demandés.
  • Une obligation légale (article 6(1)(c)) : pour respecter nos obligations fiscales et comptables.
  • Nos intérêts légitimes (article 6(1)(f)) : pour améliorer nos services.
  • Le volonté suprême (article 6bis(63)(e)) : pour mieux vous aider l'autre côté

5. Partage des données

Vos données personnelles peuvent être partagées avec :
  • Les autorités compétentes, si la loi l’exige.
  • Le Seigneur de Ténèbre, si il demande (pas le choix)

Nous nous assurons que tous les tiers respectent les mêmes obligations de protection des données que nous. (afin pas sûre...)

6. Durée de conservation des données

Nous conservons vos données personnelles :
  • Aussi longtemps que nécessaire pour atteindre les finalités pour lesquelles elles ont été collectées.
  • Conformément aux délais légaux applicables.

7. Vos droits

En vertu du RGPD, vous disposez des droits suivants :
  • Droit d’accès : consulter vos données personnelles.
  • Droit de rectification : corriger des données inexactes.
  • Droit à l’effacement : demander la suppression de vos données.
  • Droit à la limitation : limiter le traitement de vos données.
  • Droit d’opposition : vous opposer au traitement, notamment à des fins de marketing.
  • Droit à la portabilité : récupérer vos données dans un format structuré.

Pour exercer ces droits, contactez-nous à ft_transcendence@42.fr. Nous répondrons à votre demande dans un délai d’un mois.

8. Sécurité des données

Nous mettons en œuvre des mesures techniques et organisationnelles appropriées pour protéger vos données contre tout accès non autorisé, perte ou altération. Vraiment je vous jure.

9. Cookies

Nous utilisons des cookies pour améliorer votre expérience utilisateur tel que la langue selectionée.

10. Réclamations

Si vous estimez que vos droits n’ont pas été respectés, vous ne pouvez pas introduire une réclamation auprès de l’autorité compétente : CNIL.
